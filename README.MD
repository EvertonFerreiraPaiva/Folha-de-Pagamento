## 📌 Sobre o Projeto

O **Holerite Digital 2026** é uma aplicação web desenvolvida para simular o cálculo de folha de pagamento, contemplando descontos de INSS, Imposto de Renda e cálculo do salário líquido.

Projeto com foco educacional, aplicando lógica de programação e manipulação de DOM com JavaScript.

---

## 🎯 Objetivo

- Praticar lógica de programação
- Trabalhar com manipulação de DOM
- Simular regras de cálculo financeiro
- Criar interface amigável

---

## 🚀 Funcionalidades

✔️ Inserção de salário bruto  
✔️ Cálculo automático do INSS  
✔️ Cálculo de IRPF  
✔️ Exibição do salário líquido  
✔️ Botão de limpar dados  

---

## 🧮 Regras de Negócio

### INSS (Progressivo)

| Faixa Salarial (R$) | Alíquota |
|--------------------|---------|
| Até 1.621,00       | 7,5%    |
| 1.621,01 a 2.902,85 | 9%     |
| 2.902,86 a 4.354,27 | 12%    |
| Acima de 4.354,27  | 14%    |

🔒 Teto máximo: **R$ 988,09**

---

### IRPF (Simplificado)

- Isento até: **R$ 5.000**
- Acima disso: **27,5% sobre o excedente**

> ⚠️ Modelo simplificado para fins acadêmicos
